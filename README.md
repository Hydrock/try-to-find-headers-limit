# Поиск максимального размера заголовков

Выполни `yarn limit`

Укажи URL

Выбери режим: ручной или автоматический

В ручном режиме, вы самостоятельно укажете размер заголовка - запрос будет только один.

В автоматическом запросе будет выполнено нескольно инкрементальных запросов для поиска максимального размера заголовков.
